# Política de Privacidad — Login Verd para la UPC

Última actualización: 23 de junio de 2026

## Datos que maneja la extensión

Login Verd almacena, **únicamente en el dispositivo del usuario**, los siguientes datos para poder iniciar sesión automáticamente en `login.upc.edu`:

- Nombre de usuario de la UPC
- Contraseña de la UPC
- Secreto TOTP (para generar el código de doble factor, 2FA)
- Nombre de dispositivo (opcional)

## Cómo se almacenan

Los datos se guardan localmente mediante la API `chrome.storage.local` del navegador. **No se transmiten, venden ni comparten con el desarrollador ni con terceros.** Nunca salen del dispositivo del usuario.

## Uso de los datos

Los datos se usan exclusivamente para rellenar el formulario de inicio de sesión y completar el segundo factor de autenticación (2FA) en `login.upc.edu`. No se utilizan para ningún otro fin, no se usan para determinar la solvencia económica ni con fines crediticios.

## Eliminación de los datos

El usuario puede borrar los datos en cualquier momento desde la página de opciones de la extensión o desinstalándola.
